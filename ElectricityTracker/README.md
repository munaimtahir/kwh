# ElectricityTracker

An Android app to track electricity consumption.

## Features

* Log electricity usage
* View usage history
* Analytics
* Notifications
* Settings

## Tech Stack

* Kotlin
* Jetpack Compose
* MVVM
* Room
* WorkManager
* Notifications
* Hilt
* Coroutines
* Flow
* Material Design 3

## Setup

1. Clone the repository
2. Open in Android Studio
3. Build and run
